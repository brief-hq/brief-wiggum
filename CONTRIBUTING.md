# Contributing to Brief Wiggum

This is an experimental project. Fork it, adapt it to your needs, and use it as a starting point.

## Pull Requests

PRs are welcome but not guaranteed a response. If you've made improvements you think others would benefit from, feel free to open a PR.

1. **Fork the repository**
2. **Create a branch** - `feature/add-python-support` or `fix/git-guard-regex`
3. **Make your changes** - Follow existing patterns
4. **Test your changes** - Run setup, verify hooks work
5. **Submit a pull request**

## Guidelines

### Skills

- Follow the existing SKILL.md format
- Include clear triggers and workflows

### Commands

- Keep commands focused on one workflow
- Document all steps clearly

### Hooks

- Test regex patterns thoroughly
- Provide helpful error messages

### Scripts

- Use `set -euo pipefail` for safety
- Add clear comments
